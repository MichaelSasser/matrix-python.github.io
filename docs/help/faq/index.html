The bridge is not and was never stable. There are a bunch of stability and moderation problems. One of our biggest critic points of the #python (IRC) is the need to be registered on libera to participate. It is not that big of a difficulty, when you used an IRC before, and you know how it works. Otherwise, you need to look it up. Then you need to find out how to you do that via the appbridge/heisenbridge. There is no GUI to do that. That is a bad user experience we don’t want.
In this community, it should be easy to join and chat. We are not interested in scaring away users because of complexity bridging with it. If you want to join #python on libera.chat , you can use the usual tools to do so, or you select “Matrix” in “libera.chat” when you click on “explore rooms”. If you are not on matrix.org, you can add that server.
